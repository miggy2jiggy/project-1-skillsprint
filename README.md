# SkillSprint

## Project Overview

SkillSprint is a responsive PHP-powered website created for CSC 4370 Project 1. The website helps users explore technology career paths, discover the skills needed for each career, and generate a personalized learning roadmap based on their interests and learning preferences.

## Features

- Multi-page PHP website
- Shared header, navigation, and footer using PHP includes
- Dynamic career information generated from PHP arrays
- Skills Library displayed using `foreach` loops
- Career Planner form using `$_POST`
- Server-side form validation
- Sticky form inputs after validation
- Personalized career roadmap
- PHP session-based welcome message
- CSS-only FAQ accordion
- Responsive design for desktop, tablet, and mobile devices
- CSS variables for reusable styling

---

## Pages

- `index.php` — Home page
- `career-tracks.php` — Technology career information
- `skills-library.php` — Dynamic skills library
- `career-planner.php` — Career planner form and personalized roadmap
- `about.php` — About page and FAQ section
